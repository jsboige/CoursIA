# Configurations Docker

Ce répertoire contient les fichiers de configuration Docker validés et sanctuarisés pour l'infrastructure ComfyUI-Qwen.

## ⚙️ Fichiers Principaux

### Docker Compose
- `docker-compose.yml` : Configuration principale validée
- `docker-compose-no-auth.yml` : Version sans authentification
- `docker-compose-simple.yml` : Configuration simplifiée

### Environnement
- `.env` : Variables d'environnement sécurisées
- `.env.example` : Template de configuration
- `.env.backup` : Sauvegarde de configuration

### Scripts Utilitaires
- Scripts de génération de tokens
- Outils de validation
- Tests de connectivité

## 🔒 Sécurité

Toutes les configurations ont été validées pour :
- ✅ Sécurisation des accès
- ✅ Isolation des réseaux
- ✅ Gestion des secrets
- ✅ Validation des permissions

## 🚀 Déploiement

Les configurations sont prêtes pour :
- Déploiement en production
- Tests de charge
- Maintenance régulière
- Sauvegarde et restauration
