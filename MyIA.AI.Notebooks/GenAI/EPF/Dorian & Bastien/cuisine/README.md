## Objectif du projet

Développer un système intelligent capable de générer des recettes culinaires en fonction des ingrédients disponibles et des préférences alimentaires.

## Technologies utilisées

- **Python** : Traitement de données, génération de texte
- **IA** : Modèles de langage pour la génération de recettes
- **Data** : Base de données d'ingrédients, allergies, préférences

## Fonctionnalités

- Génération de recettes personnalisées
- Suggestions d'alternatives d'ingrédients
- Gestion des allergies et restrictions alimentaires
- Planification automatique de menus
