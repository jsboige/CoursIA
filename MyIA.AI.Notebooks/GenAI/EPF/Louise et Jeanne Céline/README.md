## Objectif du projet

Créer un chatbot capable d'assister les patients en répondant à leurs questions médicales avec précision et empathie, tout en maintenant la sécurité et la confidentialité.

## Technologies utilisées

- **Python** : Développement de l'interface
- **IA** : Modèles de langage spécialisés en médecine
- **Sécurité** : Protection des données sensibles
- **API** : Intégration avec bases de données médicales

## Fonctionnalités

- Questions-réponses médicales
- Analyse des symptômes (basique)
- Orientation vers les spécialistes
- Historique des conversations

## Prérequis

```bash
pip install openai pandas scikit-learn
```

## Enjeux

- **Confidentialité** : Protection des données médicales
- **Précision** : Réponses médicales fiables
- **Empathie** : Communication humaine
- **Sécurité** : Pas de diagnostic automatique

## Résultats attendus

- Chatbot médical fonctionnel
- Système de qualité et vérification
- Interface sécurisée et intuitive
