# Projet Exemple - Gestion Utilisateurs

Petit projet Python pour démontrer les fonctionnalités de Claude Code.

## Installation

```bash
pip install -r requirements.txt
```

## Utilisation

```bash
python src/main.py
```

## Tests

```bash
pytest tests/
```

## Structure

```
src/
├── main.py          # Point d'entrée
├── utils.py         # Fonctions utilitaires
└── models/
    └── user.py      # Classe User
tests/
└── test_utils.py    # Tests unitaires
```
