# Demo 1: Analyse de Données Scriptée et Reproductible

Cette démo vous guide dans l'utilisation de Roo pour effectuer une analyse de données reproductible en créant un script d'analyse et un rapport basé sur ses résultats.

## Objectif

Apprendre à utiliser Roo pour :
- **Créer un script d'analyse** (Python ou PowerShell) pour traiter un ensemble de données.
- **Générer des résultats** (fichiers CSV, graphiques) à partir du script.
- **Rédiger un rapport d'analyse** (`rapport_analyse.md`) qui interprète les résultats produits par le script.

## Contexte

Vous êtes analyste de données. Votre mission est d'analyser le fichier `ressources/exemple-donnees-ventes.csv` pour en extraire des informations clés. Votre analyse doit être **reproductible**, ce qui signifie que n'importe quel membre de votre équipe doit pouvoir exécuter votre script et obtenir les mêmes résultats.

## Instructions

### Étape 1 : Création du Script d'Analyse

1.  **Créez un nouveau fichier** pour votre script (par exemple, `analyse_ventes.py` ou `analyse_ventes.ps1`).
2.  **Demandez à Roo de vous aider à écrire le code** pour :
    *   Charger le fichier `ressources/exemple-donnees-ventes.csv`.
    *   Calculer des statistiques descriptives de base (moyenne, médiane, total des ventes).
    *   Identifier le produit le plus vendu.
    *   Identifier le mois avec le chiffre d'affaires le plus élevé.
    *   *Optionnel :* Générer un graphique simple (ex: ventes par mois) et le sauvegarder en tant qu'image.
3.  Assurez-vous que votre script **affiche clairement les résultats** dans la console ou les sauvegarde dans un fichier texte.

### Étape 2 : Rédaction du Rapport d'Analyse

1.  **Exécutez votre script** et copiez les résultats qu'il génère.
2.  **Créez un fichier `rapport_analyse.md`**.
3.  Dans ce rapport, **présentez les résultats obtenus** à l'étape précédente.
4.  **Demandez à Roo de vous aider à interpréter ces résultats** et à rédiger des conclusions claires et concises. Le rapport doit expliquer ce que signifient les chiffres (par exemple, "Le produit X est le plus performant, ce qui suggère une forte demande client pour cette catégorie.").

## Livrables Attendus

1.  **Le script d'analyse** (ex: `analyse_ventes.py`).
2.  **Le rapport d'analyse** (`rapport_analyse.md`) contenant l'interprétation des résultats générés par le script.

## Fonctionnalités Mises en Avant

-   **Génération de code**: Utiliser l'IA pour écrire des scripts d'analyse fonctionnels.
-   **Analyse de données reproductible**: Créer des solutions qui peuvent être vérifiées et réutilisées.
-   **Interprétation assistée par IA**: Transformer des données brutes en insights métier pertinents.
-   **Génération de rapports**: Structurer une analyse dans un document clair et professionnel.
