# Introduction à la Solution de Matching CV

Bienvenue dans la documentation du projet de matching CV. Ce document sert de point de départ pour comprendre les objectifs, le fonctionnement et la structure de cette application.

## 1. Quel est l'objectif de ce projet ?

L'objectif principal est de fournir une plateforme d'expérimentation pour comparer l'efficacité de différents algorithmes de matching entre les CV de consultants et les fiches de poste. Le projet vise à aller au-delà de la simple correspondance de mots-clés en explorant des approches sémantiques plus fines.

## 3. À qui s'adresse ce projet ?

Ce projet est destiné à plusieurs types d'acteurs :

-   **Développeurs et Data Scientists** : Pour expérimenter avec des algorithmes de matching et des technologies sémantiques.
-   **Chefs de Projet Technique** : Pour évaluer une solution de matching CV robuste et testable.
-   **Recruteurs Techniques** : Pour comprendre comment l'IA peut aider à identifier les meilleurs profils.
