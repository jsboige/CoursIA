### `ema-grid-heatmap.png` — Dual-panel heatmaps Sharpe + MaxDD Grid Search EMA

**Alt-text (FR)** *(c.449 reformulé CONTENT-driven)* : **Deux heatmaps côte à côte** (1539×490, format `plt.subplots(1, 2, figsize=(16, 5))`). Axes partagés : X = Slow EMA period (30/40/50/60/100), Y = Fast EMA period (8/10/12/15/20). Palette divergente RdYlGn (rouge=mauvais, vert=bon). **Gauche — « Sharpe ratio - Grid Search EMA (2015-2026) »** : valeurs 0.62 → 0.92. **Best Sharpe = 0.92** à fast=8 / slow=100 (coin haut-droite). Autres configurations : 0.85 (8/30), 0.82 (10/100), 0.78 (12/60), 0.74 (20/50 baseline), 0.62 (8/30). **Droite — « MaxDD (vert = plus petit) »** : valeurs 14.7% → 28.7%. **Best MaxDD = 14.7%** à fast=10 / slow=50 (vert vif au centre-haut). Pire DD = 28.7% à fast=8 / slow=30 (rouge vif, signaux trop rapides = whipsaws). **Pas de sweet spot unique** : la configuration qui maximise Sharpe (8/100) n'est pas celle qui minimise DD (10/50). Trade-off classique risk/return.

**Contenu réel vérifié** (audit visuel MiniMax M3, c.449) : dual-panel heatmaps annotées avec valeurs dans chaque cellule, légendes d'axes visibles (Slow EMA / Fast EMA), titres sur chaque subplot. **Alt-text précédent** « Heatmap du Sharpe ratio en fonction des périodes EMA (fast vs slow) » était **TITLES-driven ET INCOMPLET** : omettait la moitié droite de la figure (MaxDD heatmap) et les conclusions clés (best Sharpe vs best MaxDD = trade-off).
