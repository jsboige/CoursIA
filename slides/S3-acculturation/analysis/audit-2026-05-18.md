## 3. Couverture vs PPTX original

**Verdict global** : Slidev **fortement enrichi** (+108%) sur la base d'un PPTX deja tres visuel. Ratio image PPTX 2.1 — verification critique que les 135 images PPTX sont preservees.

### Points forts heritage PPTX
- **Couverture transversale exceptionnelle** : DNN, RNN, LSTM, GANs, SMT, ontologies, smart contracts (deck d'acculturation generale)
- **Branding personnel** : Logos DNN, PKP, MyIA (intro)
- **Recence** : janvier 2025 — captures les developpements LLM/GenAI recents
- **Splits Slidev** : Developpement, Strategies exploration, Theorie des jeux, Caracteristiques, Applications — tous splits a/b pour reduire densite

### Risques residuels (verification visuelle requise — niveau de risque le plus eleve des 3 decks)
- **Inventaire images** : 135 PNG dans `slides/S3-acculturation/images/` ? Verifier count + integration `<img>` dans `slides.md`. Toute perte d'image = regression pedagogique.
- **Layouts two-cols nombreux** (38 occurrences) : colonnes droites = `images` ou `code` ? Aucune ne doit etre vide.
- **Slide "Extensions 2010+/2015+/2020+"** : chronologie cle, verifier dates et exemples concrets (BERT, GPT, ChatGPT, multimodal).
- **Overlap argumentation S1 + Phase E NLP** : double exposure intentionnelle ?
- **Slide PPTX 1 (intro branding personnel)** : Logos MyIA + DNN + PKP doivent etre presents dans Slidev cover ou slide 1.
- **Slide Quiz** (3 sections Quiz dividers) : interactivite preservee ou simple placeholder ?
